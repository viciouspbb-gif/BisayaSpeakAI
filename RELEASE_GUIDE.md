4. **署名鍵を設定**
   - Key store path: `C:/Users/katsunori/CascadeProjects/BisayaSpeakAI/bisaya-speak-ai.jks`

### 4.2 ストアの設定

#### メインのストアの掲載情報

1. **アプリ名**
   ```
   Bisaya Speak AI - ビサヤ語発音練習
   ```

2. **簡単な説明**（80文字以内）
   ```
   AIでビサヤ語の発音を学ぼう！初心者から上級者まで、楽しく効果的に学習できます。
   ```

3. **詳細な説明**（4000文字以内）
   - `PLAY_STORE_LISTING.md` の内容をコピー

4. **アプリアイコン**
   - サイズ: 512 x 512 px
   - 形式: PNG（透過なし）

     5. 詳細フィードバック画面
     6. 参照音声再生画面

7. **カテゴリ**
   - アプリ: `教育`
   - タグ: `言語学習`

8. **連絡先の詳細**
   - メールアドレス: あなたのメールアドレス

## ❓ トラブルシューティング

### APKアップロードエラー

**エラー**: 署名が無効
- **解決**: 正しい署名鍵を使用しているか確認

**エラー**: バージョンコードが重複
- **解決**: `build.gradle.kts` の `versionCode` を増やす

### 審査で却下された

**理由**: プライバシーポリシーが不十分
- **解決**: プライバシーポリシーを詳細に記載

**理由**: 権限の説明が不足
- **解決**: マイク権限の使用目的を明確に説明

### アプリが見つからない

**原因**: 審査中
- **解決**: 審査完了を待つ

**原因**: 地域制限
- **解決**: 公開地域を確認

1. **署名ファイルの正解パス**  
   `C:\Bisaya_Final\bisaya-speak-ai.jks`

2. **パスワードの法則**  
   すべて `Bisaya2025` を使用（`storePassword` / `keyPassword` 共通）。変更する場合は必ず `local.properties` を更新し、コードへ直書きしない。
