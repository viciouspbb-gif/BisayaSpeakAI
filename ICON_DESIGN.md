# Bisaya Speak AI - アイコンデザイン仕様

Google Play Store用のアプリアイコンデザイン仕様書です。

## 🎨 デザインコンセプト

### コアアイデア
**「AI × 会話 × 音声」を融合したモダンなアイコン**

- **吹き出し**: 会話・コミュニケーションを象徴
- **回路パターン/音声波形**: AI技術と音声分析を表現
- **シンプル**: 一目で認識できる明快なデザイン

## 🌈 カラーパレット

### メインカラー: ターコイズブルー

**Primary Blue**
- HEX: `#00BCD4`
- RGB: `(0, 188, 212)`
- CMYK: `(100, 11, 0, 17)`
- 用途: 吹き出しの背景、メインカラー

**Dark Blue**
- HEX: `#0097A7`
- RGB: `(0, 151, 167)`
- 用途: 吹き出しのアウトライン、影

### アクセントカラー: 明るい緑

**Accent Green**
- HEX: `#4CAF50`
- RGB: `(76, 175, 80)`
- CMYK: `(57, 0, 54, 31)`
- 用途: 回路ノード、ハイライト

**Light Green**
- HEX: `#8BC34A`
- RGB: `(139, 195, 74)`
- 用途: グラデーション、アクセント

### 背景・補助色

**White**
- HEX: `#FFFFFF`
- RGB: `(255, 255, 255)`
- 用途: 背景、吹き出し内部

**Light Gray**
- HEX: `#F5F5F5`
- RGB: `(245, 245, 245)`
- 用途: 影、深度表現

## 📐 サイズ仕様

### Google Play Store要件

#### 必須サイズ
1. **512 x 512 px** (高解像度アイコン)
   - 形式: PNG（32-bit）
   - 最大サイズ: 1024 KB
   - 透過: 不可（背景必須）

#### 推奨追加サイズ
2. **192 x 192 px** (xxxhdpi)
3. **144 x 144 px** (xxhdpi)
4. **96 x 96 px** (xhdpi)
5. **72 x 72 px** (hdpi)
6. **48 x 48 px** (mdpi)

### セーフゾーン
- 外側から **10%** のマージン
- 512px の場合: 51px のマージン
- 実際の描画エリア: 410 x 410 px

## 📝 作成手順（Figmaの場合）

### ステップ1: 新規ファイル作成
1. Figmaを開く
2. 新規ファイル作成
3. フレームサイズ: 512 x 512 px

### ステップ2: 背景作成
1. 長方形ツール（R）
2. サイズ: 512 x 512 px
3. 塗り: 白 (#FFFFFF)

### ステップ3: 吹き出し作成
1. 長方形ツール（R）
2. サイズ: 350 x 280 px
3. 位置: 中央（X: 81, Y: 81）
4. 角の丸み: 40 px
5. 塗り: ターコイズブルー (#00BCD4)
6. 枠線: 8 px, ダークブルー (#0097A7)

### ステップ4: 吹き出しの尾作成
1. 三角形ツール
2. サイズ: 60 x 40 px
3. 位置: 中央下部
4. 塗り: ターコイズブルー (#00BCD4)
5. 枠線: 8 px, ダークブルー (#0097A7)

### ステップ5: 内部の白背景
1. 長方形ツール（R）
2. サイズ: 334 x 264 px（吹き出し内部）
3. 角の丸み: 32 px
4. 塗り: 白 (#FFFFFF)

### ステップ6: 回路ノード作成
1. 円ツール（O）
2. サイズ: 30 x 30 px
3. 4つ作成（四隅）
4. 塗り: 明るい緑 (#4CAF50)

### ステップ7: 接続線作成
1. 線ツール（L）
2. 太さ: 6 px
3. 色: 明るい緑 (#4CAF50)
4. ノード間を接続

### ステップ8: 音声波形作成
1. ペンツール（P）
2. 波形を描画（5-7本）
3. 太さ: 4 px
4. 色: 明るい緑 (#4CAF50)
5. 中央に配置

### ステップ9: エクスポート
1. フレームを選択
2. Export → PNG
3. サイズ: 512 x 512 px
4. 品質: 最高

## 📱 適応型アイコン（Android）

### Adaptive Icon要件

#### レイヤー構成
1. **Foreground Layer**: メインデザイン
2. **Background Layer**: 背景色

#### サイズ
- 108 x 108 dp
- セーフゾーン: 中央 72 x 72 dp

#### 実装
```xml
<!-- res/mipmap-anydpi-v26/ic_launcher.xml -->
<?xml version="1.0" encoding="utf-8"?>
<adaptive-icon xmlns:android="http://schemas.android.com/apk/res/android">
    <background android:drawable="@color/ic_launcher_background"/>
    <foreground android:drawable="@drawable/ic_launcher_foreground"/>
</adaptive-icon>
```
